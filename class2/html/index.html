border - Borde

Aplica un borde al elemento en cuestion. Esto es un 'shortcut' para definir el grosor, el tipo de linea y color.

Estilo:
border: 1px solid black;

border-left - Borde Izquierdo

Aplica un borde al elemento en cuestion, pero solo el lado izquierdo. Esto es un 'shortcut' para definir el grosor, el tipo de linea y color.

Estilo:
border-left: 1px solid black;

border-right - Borde Derecho

Aplica un borde al elemento en cuestion, pero solo el lado derecho. Esto es un 'shortcut' para definir el grosor, el tipo de linea y color.

Estilo:
border-right: 1px solid black;

border-bottom - Borde Inferior

Aplica un borde al elemento en cuestion, pero solo en el borde inferior. Esto es un 'shortcut' para definir el grosor, el tipo de linea y color.

Estilo:
border-bottom: 1px solid black;

border-top - Borde Superior

Aplica un borde al elemento en cuestion, pero solo del borde superior. Esto es un 'shortcut' para definir el grosor, el tipo de linea y color.

Estilo:
border-top: 1px solid black;

border-radius - Borde Radial

Permite recortar las esquinas del borde.

Estilo:
border-radius: 10px;

background-image - Imagen de Fondo

Permite poner una imagen de fondo.

Estilo:
background-image : linear-gradient(rgb(237, 160, 160), rgb(250, 24, 24));
